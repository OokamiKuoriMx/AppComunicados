# AppComunicados - Google Apps Script

Sistema de gestión de comunicados desarrollado en Google Apps Script.

## Estructura del Proyecto

```
appScript/
├── code.gs                    # Código principal del servidor
├── crud.gs                    # Operaciones CRUD generales
├── comunicados.crud.gs        # CRUD específico para comunicados
├── datosGenerales.crud.gs     # CRUD para datos generales
├── utilidades.gs              # Funciones utilitarias
├── index.html                 # Página principal
├── home.html                  # Vista de inicio
├── header.html                # Componente de cabecera
├── sidebar.html               # Barra lateral de navegación
├── styles.html                # Estilos CSS
├── script.html                # Scripts JavaScript comunes
├── comunicados.html           # Vista de comunicados
├── comunicados.js.html        # JavaScript para comunicados
├── comunicado_detalle.html    # Vista detalle de comunicado
├── comunicado_detalle.js.html # JavaScript para detalle
├── cuentas.html               # Vista de cuentas
└── cuentas.js.html            # JavaScript para cuentas
```

## Tecnologías

- Google Apps Script
- HTML/CSS/JavaScript
- Bootstrap
- Google Sheets como base de datos

## Despliegue

Este proyecto se despliega como una Web App en Google Apps Script.
